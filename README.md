## Development

### Prerequisites
- **Node.js**
- Text editor or IDE

### Local Development

1. **Clone the repository**:
   ```bash
   git clone https://github.com/c2crobotics/c2crobotics.github.io.git
   ```

2. **Install dependencies**:
   ```bash
   # Using npm
   npm install
   
   # Using yarn
   yarn install
   
   # Using pnpm
   pnpm install
   ```

3. **Start the development server**:
   ```bash
   # Using npm
   npm run dev
   
   # Using yarn
   yarn dev
   
   # Using pnpm
   pnpm dev
   ```

4. **Navigate to `http://localhost:3000`**

### Build for Production

```bash
npm run build
```
